新物种爆炸.吴声商业方法发布2019

发表于 2019-08-05 分类于 趋势 评论数： 热度：

人物介绍

吴声:场景实验室创始人,中国互联网原创商业思想的重要研究者和实践者。著有《场景革命：重构人与商业的连接》 《超级IP：互联网新物种方法论》 和《新物种爆炸：认知升级时代的新商业思维》 三部商业作品，对中国新经济企业影响深远。

吴声商业方法发布2019

2019年8月4日下午14:00时，今年的“新物种爆炸·吴声商业方法发布2019”在北京751D·PARK 79罐如期举行。在这个3小时的演讲上吴声老师对数字商业新时代这个主题进行了个人观点的阐述并对未来的商业模式进行了大胆预测。（虽然我听的并不太懂）

数字商业正年轻

演讲开场吴声指出：随着00后这一波数字原住民的正式入场，数字商业正在被重新定义，正因为如此数字商业正年轻。

对于00后这波数字原住民的特征，吴声老师总结如下：

特征1：天生边缘——非主流的主流性；

特征2：自组织——随时发起的连接性；

特征3：开放协作——维基百科式的开源精神。

这也是这次演讲的开场，并列举了：

以数字化咖啡为代表的——瑞幸咖啡，通过成本、体验、效率的结构性变化，改造我们习以为常的每一种消费常态

以数字化生活为代表的——小米有品，通过新、奇、酷的理念，提升男性甚至女性用户对科技、数码产品的兴趣。

以数字汽车为代表的——拜腾汽车，通过线下体验店同汽车大平台的不断交互，持续将用户体验向前推进。

以数字体育为代表的——NBA，通过VR直播，360度环场摄影，科技篮球鞋，将赛事传变成了科技的实验地。

他认为边缘、速度、离散，构成了我们所认知年轻商业的规则，所以不要因为自己非主流化而惴惴不安，也不要认为自己边缘而感到孤独。

而是否有快速连接和介入的能力，是否能随时感知到商业最细小、最微弱的变化，感知到年轻的消费形态和消费精神，是我们今天互联网从业者和企业创始人所面对的全新挑战和考验。

年轻商业的算法

在演讲中，吴声老师将年轻的商业算法特点归纳为为：小、多、轻。

小 是指来自边缘的持续连接，是一种感受。

1、用小数据捕捉小需求

——买菜这件小事上列举了叮咚买菜，把握刚需、痛点、高频的用户买菜需求，重定义买菜场景请求，完成一种替代和优化效率的建模。

——睡眠这件小事上列举了慕思，重定义用户睡眠场景，AI+大数据+睡眠 不断迭代新的数字化睡眠产品。

2、用小数据解决小问题

——手机充电这件小问题上列举了云充吧，抓住用户随时随地刷屏，随时随地吃鸡，随时随地里程焦虑的痛点。

3、用小数据驱动大场景

——人工智能行业列举了碧桂园 博智林机器人，大疆的教育机器人“机甲大师”

——金融行业列举了百信银行，以“场景+”开放银行生态战略的金融新SaaS。个性化的场景定制服务，为数字化金融转型。

**小数据是场景数据，谁更能驾驭小数据，谁就更有机会定义年轻场景，更有机会定义新品类。

比如：日本9hours 胶囊公寓+咖啡，它的逻辑特别简单——最好的睡眠舱、最好的咖啡，但最重要的是“共享Desk”。

日本文喫，付费制阅读空间。重新定义了“知识书店”的理念。

多 是指用户离散的处处触点

1、在用户是离散的、漂移的、不确定的情况下，“多”意味着今天品牌没有办法通过爆品一招鲜吃遍天，需要更多灵活的解决方案。

——例举了MUJI到REMUJI“不多余”的朴素品牌理念。

2、有多少率性需要，就有多少新鲜方案。

——列举了生于AloT时代的小米，在九年里面不断进化、生长，最终成长为全球联网设备最多的AIoT平台。

3、多等于泛在传感，无处不在、无微不至。

——列举了滴滴汽车之后的首约汽车可以针对特定的会务场景，通过独特的安保能力和国宾车队的价值去建立高效、敏捷的响应，成为场景定制的智慧出行平台。

——列举了分众传媒已经做到精准场景的数字化打通。

——列举了买手零售平台TJX，通过数据赋能买手。

4、多场景涌现新场景。这意味着随时可能发起的自组织连接，多场景互相塑造，共生出全新的场景。

——列举言几又桐庐book&bed、度刻、阿那亚、Showfields 来阐述多品牌会形成完整的解决方案，更重要的是数字化的反馈能力。

——列举红星美凯龙，已颠覆传统家具卖场，转型成为越来越热衷于“造场”的超级流量场，赋能每个商家打造全场景全链路全周期的营销场。

——列举腾讯智慧零售 以多触点的数字化能力，让连接创造价值。

轻 是累进数据的共享授权，而云是轻的终极词汇

1、吴声老师阐述了谁拥有云，谁就拿到了抵达场景深处的门票。

——列举了腾讯云，微盟，企业上云已是必然的趋势。

2、表达了我们所说的轻一定是需要有人帮助我们“举重”，我们的“若轻”，是因为长期孜孜不倦产业链效率的深耕，完成重力能力的表达而形成的交付。

3、同时将数据中台、IP授权和资本效率这三样定义为“下沉协作方法论”。它不是地域概念，而是我们应该进入数字化尚未改造的领域。

——列举了大搜车从完善的SaaS到云智能的深化。

边缘、离散、授权，小数据、泛在传感、场景云，它们构成了场景算法，是系统化探寻年轻商业可能性的底层能力模型。

年轻商业6大预测

演讲的最后，吴声老师对年轻的商业进行了大胆的6项预测：

主要在以下四个方面体现：

第一，逆向生产。过剩材料循环使用所主导的设计思维，释放新的商业价值。列举爱马仕将货箱、围巾、皮革残片等再利用创造新作品如家居摆件、小配饰。

第二，产品循环咨询服务。判断产品循环生态的咨询服务将成为全新的社会分工。

第三，时尚环保供应链设计。“设计”不再是具体的物质结果，而是从原材料到生产，到回收的循环体系。类似于一个独立自洽的循环小生态系统。

第四，可持续消费教育。可持续消费理念和乐活方式将成为广域教育新学科。

E是环境（Environment），S是社会（Social Responsibility），G是治理（Corporate Governance）。企业的ESG数据将成为投资人的重要考量。同时每个企业都不得不成为社会性企业。

主要在以下四个方面体现：

第一，新的企业内部量化指标。包括动物友好度（组织能力，员工关系），内部碳排放（进化商业模式），女董事比列（体现公平、良性的竞争关系）三个指标。

第二，新的可持续竞争规则。丛林法则已不在适合所有的企业，企业应该有一个新的起点、新的能力、新的范式和新的道德准则。

第三，数据真实性产业。评估数据真实性的第三方机构有可能成为真实的服务体，其建立的评估机制将为企业提供常态化和品类化的服务。

第四，算法价值观评估师。算法价值观评估师将和精算师、会计师、律师一样成为社会和企业重要的职位，是算法时代的人类陪审团。或许将由人类学家、社会学家、哲学家等共同组成。

个性化审美、无性别设计愈发成为大趋势，美学秩序被怪美重新定义。

主要在以下四个方面体现：

第一，编码时代的唯一性产品。小量订单、3D打印、“一物一码”使唯一性成为常态，未来的产品第一是产品解决方案，第二是个性化解决方案，第三是基于数据建模的弹性供应链的一种输出能力。

第二，企业级美育服务。包括效率工具、办公室空间、创始人审美的企业级美育服务品类化。创始人的审美能力，会成为企业能否持续生存、发展的要素保障。长期做美育服务的人，属于你的时代到来了。

第三，参与式环境设计。这是一种用户参与和社交网络赋能的环境组合与体验设计，列举了Moxy酒店。

第四，视觉表达议题化。不论是字体还是颜色，纹理还是材质，都能成为新品牌机会和新商业模式。

主要在以下四个方面体现：

第一，新技术的内容。吴声老师提出了5G-KOL、8K内容产品的概念以及车联网内容服务。现在需要IP大量的内容供给，为内容制作者提供了一个崭新的赛道。

第二，零门槛艺术家。AI创作工具的日常化和娱乐化，让人人都可以成为艺术家。只要你们有好的想法，策展的经理都会配合你完成个人化的独特策展。

第三，经典创作的AI传承。AI技术环境下的可以让经典IP重生。列举了新版《狮子王》。

第四，虚拟IP机器人。那些你熟悉的虚拟IP，会以虚拟IP机器人的形态成为更真实的日常存在。比如日本的超音。

老牌技术巨头实力犹存，以旧能力优势向新技术延展，在新场景实现旧技术的应用。

主要在以下四个方面体现：

第一，基因技术分子化。列举人造肉公司，基因技术降维日常应用。

第二，数字镜像服务。数字孪生与XR生活普及，依赖于越来越高速的数据传输能力。

第三，农田前置仓化。城市立体农场与便利店农田将会成为日常，你可以在便利店里摘取一根新鲜的韭菜。

第四，数字化建筑推动城市更新。数字媒体技术、灯光设计、光影策展，激活建筑、焕新社区，重塑城市建筑与城市生活的共生关系。

列举了《自然》杂志以封面文章形式发表了清华大学类脑计算研究中心施路平教授团队的相关论文。表明我们正在进入架构时代，选择了什么样的架构就选择了什么样的可能性和方向。

主要在以下四个方面体现：

第一，开发架构沟通机制。未来开发架构间的沟通协议和服务标准是真正的大产业机会。列举了孙正义收购ARM是架构授权公司，意义在于没有ARM芯片就没有研发基础。

第二，物联网OS联盟。不只是设备相连，而是不同场景的物联网OS协议许可，包括车联网、智慧家居、智慧城市、市政公用设施等等。未来万物互联时代竞争最重要的将是不同场景物联网的协议层。

第三，从专利许可到服务许可。未来的许可将是技术专利、架构标准、服务体系的许可生态。

第四，用户许可：话语体系架构决定商业生态位。企业应该有新的表达，这种表达的考验在于用户心智会决定商业的步伐能够走多远。掌握什么样的话语体系，就会成就什么样的商业价值。

最后“2019新物种实验计划”发布 就在6大趋势预测完后落下帷幕 。

一切商业皆为进化，一切进化皆为实验。

听完吴声老师的演讲，自己还是比较的困惑，特别是对未来年轻商业6大趋势的预测，当然了这些毕竟只是一种大胆的预测，能否实现也是个未知数。不过想到现如今社会发展的速度之快，相信未来的某一天，部分的预测趋势会实现吧。
