引言

Hexo是一个快速、简洁且高效的博客框架，能够快速生成静态页面，只需一条指令即可部署到 GitHub Pages, Heroku 或其他网站，支持 GitHub Flavored Markdown 的所有功能，同时还拥有强大的插件系统。

为了增加博客站点的趣味性，这次主要介绍 NexT 7.2.0 中添加视频。
